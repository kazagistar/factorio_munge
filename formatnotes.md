# Wires

Formatted as an array, which is [EntityIdA, SlotIdA, EntityIdB, SlotIDB]. Constant combinators have slots 1/2 as output slots, not 3/4

## Slots:

1. red input
2. green input
3. red output
4. green output
5. power
6. second power (swich only)
